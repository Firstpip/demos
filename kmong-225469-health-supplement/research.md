# 유사 서비스 리서치: 맞춤형 건기식 추천 플랫폼

## 1. 핏타민 (Fitamin) — fitamin.kr

**서비스 유형**: 약사 상담 기반 맞춤 영양제 설계/구독 플랫폼
**차별점**: 30명+ 전문 약사진의 1:1 상담, 건강검진 데이터 분석

### 핵심 기능
- 건강설문(4000편) + 건강검진 데이터 + 처방약 분석
- 약사 1:1 상담: 채팅/화상 (평생 무료)
- 약물 상호작용 체크: 약사가 직접 확인
- 50여가지 성분 기반 맞춤 설계
- 월 정기구독: 섭취 리워드 30%, 3개월 재상담
- 섭취 기록: 매일 1% 할인 적립, 최대 30% 할인

### 사용자 플로우
앱 다운로드 > 건강설문 > 건강검진 데이터 연동 > 약사 상담 > 맞춤 영양제 설계 > 구독 결제 > 배송 > 섭취 기록/관리

### 에너링거와의 비교
| 항목 | 핏타민 | 에너링거 |
|------|--------|---------|
| DTC 유전자검사 | 미제공 | 핵심 차별점 |
| 건강검진 연동 | O | O (CODEF API) |
| 전문가 상담 | 약사 100% | 약사 + 의사 |
| R/S 수익분배 | 확인불가 | 의사 추천 리워드 |

### 데모 참고 포인트
- 건강검진 데이터 기반 분석 결과 UI
- 약사 프로필 갤러리 + 누적 상담 건수 표시
- 섭취 기록/리워드 UX

## 2. 필리 (Pilly) — pilly.kr

**서비스 유형**: AI 기반 맞춤 영양제 추천/정기구독
**사용자 수**: 1,057,618명

### 핵심 기능
- AI 영양제 추천: 65,535가지 경우의 수, 2,300+ 논문 기반
- 건강설문 기반 (DTC/건강검진 연동 없음)
- 정기구독: 해지/변경/제품변경 상시 가능. 평균 12회(약 1년) 지속
- 필리케어: 복약 관리 (알림, 챙겨먹기)

### 데모 참고 포인트
- 설문 > 결과 > 추천 플로우 UX
- 정기구독 관리 UX (해지/변경/연기)
- 복약 관리 알림 UI

## 3. 젠톡 (GenTok) — macrogen.com/gentok

**서비스 유형**: DTC 유전자검사 + 마이크로바이옴 검사 플랫폼
**운영사**: 마크로젠 (코스닥 상장)

### 핵심 기능
- 유전자 검사 129종: 영양소(35), 운동(10), 피부/모발(18), 식습관(21), 개인특성(23), 건강관리(22)
- 마이크로바이옴 검사: 장 건강, 구강
- AI 챗봇 유전상담
- 결과: PDF + 앱 기반 시각화

### 한계
- 건강기능식품 직접 추천/판매 없음 (검사까지만)
- 전문가 1:1 상담 없음 (AI 챗봇만)
- 이커머스/구독 없음

### 데모 참고 포인트
- 유전자 검사 항목 분류 체계
- 유전자 결과 시각화 UI (항목별 위험도 등급)
- 검사 키트 신청 > 검체 발송 > 결과 확인 플로우

## 5. 디자인 시스템 (기획서 와이어프레임 기준)

> 기획서(19p)에 PC/모바일 와이어프레임이 상세하게 포함되어 있으므로 기획서 디자인을 최대한 따름.

### 기획서에서 확정된 디자인 요소
- **브랜드명**: ENER RINGER (좌상단 로고)
- **PC 네비게이션**: DTC소개 | DTC 유전자검사 | 맞춤형건강기능식품 (p10)
- **PC 우측**: 로그인 | 회원가입 | 고객센터 | 주문배송 | 앱 다운로드(iOS/Android)
- **모바일 하단 탭바**: 홈(집), 마켓(장바구니), 내결과(문서), 상담(말풍선), MY(사람들) (p12)
- **메인 히어로**: "나만을 위한 완벽한 영양 배합, 의사의 진단에서 시작됩니다" + DNA 시각화 영역 (p10)
- **추가 배너 3종**: "나만을 위한 건강식품, 의사가 직접 진단합니다" / "유전자가 알려주는 나만의 건강 레시피" / "영양제, 아무거나 드시고 계신가요?" (p10)
- **DTC 서비스 페이지**: DTC유전자 결과 불러오기 / 건강검진 기록 불러오기 / 설문조사로 추천 받기 (p13)
- **건강 리포트**: 원형 건강점수 게이지(55점/경고), 건강해요/주의가 필요해요/질환이 의심돼요 3등급, 추천 성분 10개 태그 표시 (p14)
- **유전자 마커 상세**: 카드형 마커별 위험(빨강)/주의(노랑)/양호(초록) 뱃지 + 변이 분석 세부 정보 + 생활습관 개선 팁 (p15)
- **건강검진 상세**: 검진항목별 드롭다운(고혈압 등) + 정상범위/검진결과/현재상태/권고사항 + 시계열 차트(2017~2025) (p17)
- **상담 페이지**: 전문의 상담 / 약사 상담 탭 전환, "잠깐! 나의 건강검진 결과는?" 배너, 약사 프로필 카드(이름, 약국, 누적 상담수) + 채팅상담/화상상담 버튼(그린 강조) (p15, 17, 18)
- **전문의 1:1 맞춤상담 섹션**: 전문의 단체 사진 + "상담하기" 버튼 + 후기 카드 4개 (p11)
- **의사 Admin**: 좌측 사이드바(대시보드/환자목록/추천제품/수수료정산/성과분석/QR설정) + 통계 카드 4개(추천환자수/구매전환율/누적수수료/정산예정일) + 환자 테이블 + 월별 수수료 차트 + 나의 추천링크/QR코드 (p19)

### 기획서에서 유추할 컬러 (와이어프레임 참조)
- **헤더/푸터**: 다크 네이비/블랙 배경 (p10 메인 히어로)
- **강조 버튼**: 그린 계열 (p15 화상상담 버튼, p17 결과지 영양제 추천 상담 버튼)
- **위험 뱃지**: 빨강 (p15 CYO17A1 위험)
- **주의 뱃지**: 노랑/오렌지 (p15 FGF5 주의)
- **양호 뱃지**: 초록 (p15 GUCY1A3 양호)
- **배경**: 화이트 + 밝은 그레이
- **Admin 사이드바**: 다크 배경 (p19)
- **Admin 통계 카드**: 컬러별 상단 보더 (p19 파랑/초록/노랑/빨강)

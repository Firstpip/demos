안녕하세요. 올려주신 프로젝트 내용을 꼼꼼히 확인했습니다.

DTC 유전자검사와 건강검진 데이터를 기반으로 맞춤형 건강기능식품을 추천하고, 약사 화상상담과 의사 R/S 수익분배 모델을 포함하는 B2B2C 이커머스 플랫폼 구축 프로젝트로 이해했습니다. 레퍼런스인 핏타민의 기능/구조를 참고하되, DTC 유전자검사 + 의사 R/S라는 차별점을 중심으로 설계합니다.

첨부하신 기획서(19페이지)를 상세히 분석했으며, 에너링거(ENER RINGER) 서비스의 핵심인 유전자 마커 12종 매핑표, 건강검진 13개 항목 매핑표, 사이트맵, 와이어프레임을 모두 반영했습니다.


핵심 요구사항 이해:

1. 데이터 연동 및 분석 - DTC 유전자검사 API 연동(12개 마커), 건강검진 CODEF API(마이데이터) 연동(13개 항목), 심층 설문 시스템, AI 추천 엔진(유전자+검진+설문 종합 > 제품 매칭)

2. 전문가 상담 시스템 - 약사 화상/채팅 1:1 상담(Agora SDK), 의사 추천 시스템(UTM/QR 발급), 복약 관리(알림+이력)

3. 리워드 및 정산 - 의사 추천 리워드(R/S), 약사 판매 수수료 자동 계산, 전문가별 정산 대시보드

4. 이커머스 - 맞춤형 건기식 마켓, 정기구독 결제, 배송/CS 관리

5. 개발 범위 - 사용자 Android/iOS 하이브리드 앱 + 마케팅 모바일웹, 약사 PC웹, 의사 PC/모바일웹, 관리자 PC웹


유사 경험:

이커머스 플랫폼을 기획부터 디자인, 개발까지 1인 풀스택으로 수행한 경험이 있습니다. Nuxt + Django + PostgreSQL 스택으로 상품 관리, 장바구니, 결제(토스페이먼츠), 주문/배송 관리, 회원 시스템, 소셜 로그인(네이버/카카오), 카카오 알림톡 연동까지 포함합니다. 본 프로젝트의 이커머스, PG 결제, 소셜 로그인, DB 설계와 직접적으로 연관됩니다.


구현 계획 (22주):

1주차 - 프로젝트 세팅, DB 설계, API 설계, 화면 설계 보완
2~3주차 - 인증/회원 시스템 (카카오 로그인, 본인인증 3종, 일반/전문가 가입)
4~5주차 - DTC 유전자검사 API 연동 + 12개 마커 결과 시각화
6~7주차 - 건강검진 CODEF API 연동 + 13개 검진항목 분석/시각화
8주차 - AI 추천 엔진 (설문 + 유전자 + 검진 종합 > 제품 매칭)
9~10주차 - 화상/채팅 상담 (Agora SDK), 예약 시스템
11~12주차 - 이커머스 (상품, 마켓, 장바구니, PG 결제, 정기구독)
13주차 - 리워드/정산 시스템 (R/S, UTM/QR, 수수료 자동계산)
14~15주차 - 전문가 영역 (약사 PC웹, 의사 PC/모바일웹)
16~17주차 - 관리자 + 고객센터 + 매거진 + 알림(FCM/APNs)
18~19주차 - React Native 앱 (웹뷰 + 네이티브 기능) + 스토어 배포
20주차 - 마케팅 랜딩 페이지
21~22주차 - 통합 테스트, QA, 보안 점검, 운영 배포, 인수인계


기술스택: Next.js 14 + Tailwind CSS + React Native(Expo) + PostgreSQL + Prisma + Redis + AWS Lightsail(서울 리전) + S3 + Agora SDK + CODEF API + 토스페이먼츠

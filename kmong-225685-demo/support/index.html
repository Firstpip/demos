무엇을 도와드릴까요?

자주 묻는 질문, 도입문의를 한 페이지에서 처리하세요.

SafeWorks는 어떤 솔루션인가요?
맞춤형 안전 대시보드, 안전 문서 디지털 작성, 실시간 위험 공유, 현장 맞춤형 안전점검표를 하나로 통합한 안전·문서 관리 솔루션입니다. 영신디엔씨가 현장에서 검증해 만들었습니다.
도입까지 얼마나 걸리나요?
평균 2주 이내 (상담 1일 + 계정 설정·장비 배포 3~5일 + 현장 교육 1~2일)가 소요됩니다.
안전 문서 템플릿은 얼마나 제공되나요?
기본 24종 (작업 허가서, 일일 점검 일지, 위험성 평가, TBM 일지, 안전 교육 일지 등)을 제공하며, 현장 특성에 맞게 항목을 추가·수정할 수 있습니다. Enterprise 플랜은 무제한 커스텀 템플릿을 지원합니다.
여러 현장을 한 계정으로 관리할 수 있나요?
Pro 플랜은 최대 5개 현장, Enterprise 플랜은 무제한 현장을 단일 대시보드에서 관리할 수 있습니다.
중대재해처벌법 대응에 활용할 수 있나요?
네, 모든 출입·안전 이벤트를 감사 로그로 저장하며, 법적 대응에 필요한 리포트를 자동 생성합니다. 관련 가이드라인은 다운로드 페이지에서 받아보실 수 있습니다.
기존 ERP/안전보건공단 시스템과 연동되나요?
Enterprise 플랜에서 국내 주요 ERP(더존, SAP 등) 및 안전보건공단 API 연동을 지원합니다.
데이터 보안은 어떻게 보장되나요?
전송·저장 구간 AES-256 암호화, ISO 27001·ISMS-P 준수, 역할 기반 권한 관리를 적용합니다.
현장에서 오프라인 상황은 어떻게 처리되나요?
오프라인 캐싱을 지원해 네트워크 연결이 끊겨도 로컬에 데이터가 저장되고, 재연결 시 자동 동기화됩니다.
지원은 어떻게 받을 수 있나요?
Starter: 이메일 지원 (영업일 24시간 내 응답), Pro: 전담 매니저 + 카카오톡 지원, Enterprise: 24/7 전화·온사이트 지원.
세금계산서 발행이 가능한가요?
결제 시 사업자 정보를 등록하시면 월말에 자동으로 세금계산서가 발행됩니다. 부가세는 별도입니다.

# 유사 서비스 리서치: 인플루언서 역직구 커머스 플랫폼

**작성일: 2026-04-02**

## 1. 글로벌 인플루언서 커머스 플랫폼

### 1-1. LTK (ShopLTK) — shopltk.com
- **포지셔닝**: 소셜 쇼핑 피드 (월 2,000만 앱 사용자)
- **페이지 구성**: 히어로 배너 + 카테고리 그리드 + 크리에이터 피드 + 크리에이터 프로필 페이지
- **핵심 기능**:
  - `liketk.it` 단축 URL 딥링크 시스템
  - 브랜드별 10-25% 커미션 (평균 16%), 주간 정산
  - Product Commission Comparison (동일 상품 브랜드별 커미션율 비교)
  - 크리에이터 대시보드 (팔로워/좋아요/매출/인게이지먼트/커미션)
  - 7,000+ 리테일러 네트워크, 외부 리테일러로 리디렉트 (자체 장바구니 없음)
- **UI/디자인**: 모노크로매틱(블랙/화이트) + 코랄 액센트(#FF7A59), Poppins/Roboto, 모바일 우선 3단계 반응형
- **참고 포인트**: 딥링크 시스템 구조, 커미션 비교 도구, 소셜 피드형 디스커버리

### 1-2. ShopMy — shopmy.us
- **포지셔닝**: 프리미엄 큐레이션 부티크 (크리에이터 "가상 부티크")
- **페이지 구성**: 크리에이터별 스토어프론트(/shop/, /collections/) + 브랜드-크리에이터 매칭(/opportunities/) + 탐색(/discover/)
- **핵심 기능**:
  - 자동 수익화: 47,000+ 브랜드 제휴 네트워크
  - 10-30% 커미션 + Opportunities(유료 브랜드 파트너십)
  - 매주 금요일 PayPal/Stripe 정산
  - Circles: 여러 크리에이터 추천 통합 조회
  - 브랜드 CRM: 인플루언서 성과/매출/콘텐츠/ROI 통합 관리
  - 6대 소셜 앱 감지 딥링크 리디렉트
- **UI/디자인**: 다크 테마(#1f1f1f) + 화이트 텍스트, DM Sans/Karla, 미니멀리스트
- **참고 포인트**: 크리에이터 스토어프론트(마이크로사이트), 브랜드 CRM 대시보드, 소셜 앱 딥링크

### 1-3. Amazon Influencer Program
- **포지셔닝**: 아마존 내 스토어프론트
- **페이지 구성**: amazon.com/shop/[username] 개인 스토어프론트 + Amazon Live
- **핵심 기능**:
  - 24시간 트래킹 쿠키 + 90일 구매 어트리뷰션 윈도우
  - 카테고리별 1-10% 커미션
  - Creator Connects: 소셜 콘텐츠에 제휴 상품 카드 삽입 + 실시간 추적
  - 아마존 내 직접 결제 (원클릭)
- **UI/디자인**: 아마존 표준 (오렌지 #FF9900, 네이비 #232F3E), Amazon Ember, 실용적
- **참고 포인트**: 어트리뷰션 윈도우 설계, 플랫폼 내 직접 결제, 크리에이터 콘텐츠 연계

## 2. 한국 인플루언서 커머스 플랫폼

### 2-1. 지그재그 (zigzag.kr) ⭐ 가장 유사한 구조
- **포지셔닝**: 패션 커머스 + 인플루언서 PICK 기획전
- **페이지 구성**: 모바일 퍼스트(max-width 600px), 하단 고정 네비게이션, 가로 스크롤 카드
- **인플루언서 연계**: "유튜버 PICK", "인플루언서 OOO PICK", "브랜드 X 셀럽" 기획전 다수
- **UI/디자인**: Pretendard 폰트(28px~12px 체계), #F5F6F8 라이트 그레이 배경, 밝고 신선한 톤
- **셀러 어드민**: partners.kakaostyle.com — 매출/고객/광고 통합 관리, 모바일 앱 제공
- **참고 포인트**: 인플루언서 PICK UI 패턴, Pretendard 한글 타이포, 하단 네비게이션, 가로 캐러셀

### 2-2. 에이블리 (a-bly.com) — 인플루언서 커머스 원조
- **포지셔닝**: 인플루언서 큐레이션 → AI 추천 쇼핑 (MAU 1,000만)
- **비즈니스 모델**: 인플루언서가 상품 선택 → 플랫폼이 물류/CS 대행 → 거래액 10% 수익 배분
- **UI/디자인**: 흰+검 모노톤, ABC Grow D 폰트, 파스텔 카테고리 아이콘 (10~20대 타겟)
- **어드민**: 셀러스(직접 관리) vs 파트너스(대행형) 이원화
- **참고 포인트**: 수수료 정산 구조, 셀러/파트너 이원화 어드민

### 2-3. 브랜디 (brandi.co.kr)
- **포지셔닝**: 패션 커머스 + "헬피" 풀필먼트 대행
- **UI/디자인**: Noto Sans KR, 핫핑크(#ff204b) 강조색, 2열~5열 반응형 그리드
- **셀러 어드민**: seller-admin.brandi.co.kr 별도 운영
- **참고 포인트**: 셀러 어드민 UI 구조, 풀필먼트 통합

## 3. 어드민/정산 대시보드 UI 패턴

### 3-1. 공통 레이아웃 패턴
- **다크 사이드바(#1a1a2e) + 라이트 콘텐츠(#f4f6f9)** 가 사실상 표준
- Bento Grid 레이아웃으로 KPI 카드 비대칭 배치
- Shopify Polaris의 Frame > Page > Layout > Card 계층 구조

### 3-2. 3종 어드민 차별화 전략

| 어드민 | 어센트 컬러 | 핵심 화면 | 참고 서비스 |
|--------|-----------|-----------|------------|
| 인플루언서 (영문) | 보라/남색(#6366f1) | 수익/EPC/전환율 KPI, 정산 타임라인, 리더보드 | ShopMy, LTK |
| 공급사 | 틸/초록(#0d9488) | 상품 IndexTable, 수수료 컨펌(승인/거부), 정산 명세서 | 에이블리 셀러스 |
| 통합 관리자 | 블루(#0d6efd) | GMV/매칭/정산/환율 관리, Reconciliation(대사) | Shopify Admin |

### 3-3. 데이터 테이블 패턴
- 텍스트 좌측정렬, 금액 우측정렬(monospace)
- hover시 행 액션 표시, 체크박스 + 벌크 액션
- 기본 10-25행 페이지네이션
- 상태 Badge: 초록=완료, 노랑=대기, 빨강=오류

### 3-4. 정산 시스템 UI
- **Step Indicator**: 매출집계 > 수수료산정 > 공급사컨펌 > 관리자승인 > 지급실행
- **정산 명세서**: Gross Sales - 반품 = Net Sales * 수수료율 = Commission + 환율 환산
- **Reconciliation**: 자동 매칭율 + 불일치 항목 테이블

## 5. 디자인 방향 종합

### 소비자 웹
- **톤앤매너**: 밝고 신뢰감 있는 라이트 테마 (지그재그 참고)
- **폰트**: Pretendard (한글) + Inter (영문)
- **컬러**: 화이트/라이트 그레이 배경 + 브랜드 포인트 컬러 1개
- **레이아웃**: 모바일 퍼스트, 하단 네비게이션, 가로 스크롤 캐러셀
- **핵심 UX**: 발견형 커머스(Discovery Commerce) — 콘텐츠 탐색 → 자연스러운 구매 전환

### 3종 어드민
- **공통**: 다크 사이드바 + 라이트 콘텐츠, shadcn/ui 기반
- **차별화**: 각 어드민별 어센트 컬러 분리 (보라/틸/블루)
- **차트**: Recharts
- **테이블**: TanStack Table 패턴 (shadcn/ui data-table)
- **아이콘**: Lucide Icons

### 기술 라이브러리
- shadcn/ui (컴포넌트)
- Recharts (차트)
- TanStack Table 패턴 (데이터 테이블)
- Lucide Icons (아이콘)

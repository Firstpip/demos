안녕하세요. 프로젝트 내용 꼼꼼히 확인했습니다.

미국 인플루언서를 통해 한국 소비재를 역직구 형태로 판매하는 커머스 플랫폼을 구축하는 프로젝트로 이해했습니다.

핵심 요구사항을 정리하면:

쇼핑몰 — 한국 소비자용 반응형 웹(모바일 최적화), 상품 목록/상세/검색/필터
인플루언서 딥링크 — 인플루언서별 고유 추천 링크 생성, 유입 경로 추적, 타 상품 구매도 해당 인플루언서 실적 귀속
브릿지 페이지 — 인플루언서 프로필 + 추천상품 소개 중간 페이지
인플루언서 대시보드(영문) — 실시간 매출, 확정 수수료, 정산 이력
공급사 어드민 — 입점상품 관리, 판매 내역, 인플루언서별 수수료 컨펌
통합 관리자 — 인플루언서-상품-공급사 매칭, 환율 설정, Payoneer 글로벌 정산
PG 결제 — 토스페이먼츠 카드/간편결제 연동
할인코드 — 인플루언서 전용 프로모션 코드
리포트 — 일자별/인플루언서별/공급사별 매출 및 정산 리포트 자동 생성
개발 플랫폼 — 소비자 PC/MO 반응형 웹 + 슈퍼 어드민 + 입점사 전용 어드민

유사한 이커머스 플랫폼을 기획, 디자인, 개발까지 1인 풀스택으로 수행한 경험이 있습니다. 토스페이먼츠 PG 결제 실전 연동, PostgreSQL 기반 상품/주문 DB 설계, 소셜 로그인/알림톡 연동 등 본 프로젝트의 핵심 기능과 직접적으로 연관됩니다.

구현 계획:
1단계(04/27~05/10, 14일) — 기획/설계: 요구사항 정의서, DB 스키마, API 설계, 와이어프레임
2단계(05/11~05/14, 4일) — 인프라: AWS Lightsail 서버, PostgreSQL, S3, CI/CD
3단계(05/15~05/21, 7일) — 인증/사용자: 회원가입/로그인, 역할 분리(소비자/인플루언서/공급사/관리자)
4단계(05/22~06/08, 18일) — 소비자 쇼핑몰: 상품/장바구니/결제(토스페이먼츠)/마이페이지
5단계(06/09~06/22, 14일) — 인플루언서: 딥링크 생성/추적, 브릿지, 대시보드(영문), 실적 귀속
6단계(06/23~07/02, 10일) — 공급사 어드민: 상품 CRUD, 주문/배송, 수수료 컨펌, 정산
7단계(07/03~07/12, 10일) — 통합 관리자: 매칭, 환율/정산, Payoneer 연동, 리포트
8단계(07/13~07/20, 8일) — 테스트/QA: 통합 테스트, 버그 수정, 성능 최적화
9단계(07/21~07/26, 5일) — 배포/인수: 운영 서버 배포, 관리자 교육, 문서 인수인계

기술스택: Next.js 15 + FastAPI + PostgreSQL + Tailwind CSS (AWS Lightsail + S3, 토스페이먼츠 + Payoneer)

## 요구사항 (원본 순서)

### 1. 쇼핑몰 및 인플루언서 유입 트래킹
1-1. 한국 소비자용 반응형 웹 (모바일 최적화)
1-2. 인플루언서 전용 링크(Deep Link) 생성 및 유입 경로 추적
1-3. 특정 링크 유입 고객의 메인몰 타 제품 구매도 해당 인플루언서 실적으로 귀속
1-4. 브릿지 페이지: 상품 상세 직연결 또는 인플루언서 프로필/추천상품 소개

### 2. 글로벌 정산 관리 시스템
2-1. 인플루언서 대시보드(영문): 실시간 매출, 확정 수수료, 정산 이력
2-2. 공급사(Vendor) 어드민: 입점상품 관리, 판매 내역, 인플루언서별 수수료 컨펌
2-3. 통합 관리자(Admin): 인플루언서-상품-공급사 매칭, 환율 설정, 글로벌 정산 API(Payoneer 등) 연동

### 3. 커머스 기본 기능
3-1. 국내 PG 결제 연동 (토스페이먼츠, 나이스페이 등) 및 간편결제
3-2. 인플루언서 전용 할인 코드/프로모션
3-3. 일자별/인플루언서별/공급사별 매출 및 정산 리포트 자동 생성

### 4. 향후 고도화 사항
4-1. 인플루언서별 마이크로사이트 독립 UI 및 상품 큐레이션 가능하도록 DB 설계
4-2. 인스타그램 API 등 SNS 콘텐츠 연동 구조적 여유

### 5. 암묵적 요구사항 (분석 기반 파악)
5-1. 회원가입/로그인 (소비자, 인플루언서, 공급사, 관리자 - 역할 분리)
5-2. 상품 목록/상세/검색/필터/정렬
5-3. 장바구니/주문/결제 흐름
5-4. 배송 관리 (주문 상태 추적)
5-5. 리뷰/평점 시스템
5-6. 상품 등록/수정/삭제 (공급사)
5-7. 카테고리 관리
5-8. 환불/반품 처리 프로세스
5-9. 알림 시스템 (주문 확인, 정산 완료 등)
5-10. 다국어 지원 (최소 한국어 + 영어: 인플루언서 대시보드 영문)

## 선정 기술스택: 조합 B "국내 최적화"

| 영역 | 기술 | 선정 이유 |
|------|------|-----------|
| 프론트엔드 | Next.js 15 + Tailwind CSS + shadcn/ui | 의뢰인 선호 스택, SSR/SEO 지원, 빠른 UI 개발 |
| 백엔드 | FastAPI (Python) | 의뢰인 선호(Django/FastAPI), 비동기 처리, 정산 로직에 Python 생태계 활용 |
| 인프라 | AWS Lightsail (서울 리전) | 국내 소비자 대상 → 서울 리전 필수, VPS 기반으로 cron/webhook 제약 없음 |
| DB | PostgreSQL (Lightsail 관리형) | 의뢰인 명시 스택, JSONB로 유연한 스키마, 인프라 통합 관리 |
| 파일 저장 | AWS S3 | Lightsail과 동일 AWS 생태계, 상품 이미지 CDN 연동 용이 |
| ORM | SQLAlchemy(백엔드) + Prisma(프론트) | 각 프레임워크 최적 ORM |
| 스타일링 | Tailwind CSS + shadcn/ui | 빠른 UI 개발, 일관된 디자인 시스템 |

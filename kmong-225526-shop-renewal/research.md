## 1. 한국 독립 쇼핑몰 레퍼런스

### 29CM (https://www.29cm.co.kr)
- **특징**: 깔끔한 그리드 레이아웃, 미니멀 타이포그래피, 여백 활용
- **참고 포인트**: 상품 카드 UI, 카테고리 필터링, 반응형 구조
- **적합도**: 높음 — 의뢰인이 원하는 "SaaS 빌더 아닌 자체 구축" 느낌의 프리미엄감

### 카페24 디자인센터 (https://d.cafe24.com)
- **특징**: 다양한 쇼핑몰 디자인 템플릿 모음
- **참고 포인트**: 기존 빌더 기반 쇼핑몰이 어떤 수준인지 파악 → 이보다 상위 수준으로 데모 구현
- **적합도**: 비교 기준으로 활용

## 3. 데모 디자인 방향 결정

### 채택할 디자인 원칙
1. **미니멀 그리드** — Toteme 스타일의 깔끔한 상품 그리드
2. **충분한 여백** — 제품 이미지가 돋보이도록
3. **제한된 컬러** — 중립색 기반 (흰색/회색/검정) + 포인트 색상 1개
4. **산세리프 폰트** — 현대적이고 깔끔한 인상
5. **반응형 우선** — 모바일 퍼스트, PC 확장

### 데모 구현 대상
1. **상품 목록 페이지** — 그리드 레이아웃 + 검색/필터
2. **상품 상세 페이지** — 이미지 갤러리 + 옵션 선택 + 장바구니 담기

## 4. 디자인 분석

> 29CM, Toteme, Nécessaire 등 레퍼런스 사이트 기반 디자인 시스템 정리

### 컬러 팔레트

| 용도 | 색상 코드 | 설명 |
|------|-----------|------|
| 배경 (기본) | `#ffffff` | 흰색 — 메인 배경 |
| 배경 (보조) | `#f5f5f5` | 연한 회색 — 섹션 구분, 카드 배경 |
| 보더/구분선 | `#e5e7eb` | 회색 — 얇은 구분선, 카드 보더 |
| 텍스트 (기본) | `#1a1a1a` | 거의 검정 — 본문, 제목 |
| 텍스트 (보조) | `#6b7280` | 중간 회색 — 부가 정보, 캡션 |
| 포인트 색상 | 최소화 | CTA 버튼, 배지 등에만 제한적 사용 |

> **원칙**: 중립색 기반으로 제품 이미지가 돋보이도록 구성. 포인트 색상은 CTA 등 핵심 인터랙션에만 사용.

### 폰트

| 항목 | 값 |
|------|-----|
| 서체 | 산세리프 (Geist) |
| 본문 | 14px |
| 제목 | 24~28px |
| 자간/행간 | 기본값, 넉넉한 행간 (1.5~1.7) |

### 레이아웃 패턴

- **최대 너비**: 1280px 중앙 정렬
- **그리드**: 2~4열 그리드 (페이지에 따라 유동적)
- **여백**: 충분한 패딩/마진으로 시각적 여유 확보
- **섹션 구분**: 배경색 교차 또는 얇은 보더 라인

### 컴포넌트 패턴

- **상품 카드**: 미니멀 — 이미지 + 상품명 + 가격, 불필요한 장식 제거
- **구분선**: 얇은 보더 (`1px solid #e5e7eb`)로 영역 분리
- **호버 효과**: `scale(1.02)` 확대 또는 텍스트 `underline` 효과
- **버튼**: 검정 배경 + 흰색 텍스트 (Primary), 보더 버튼 (Secondary)
- **입력 필드**: 얇은 보더, 라운드 최소 (`rounded-md`)

### 반응형 전략

| 뷰포트 | 그리드 열 수 | 비고 |
|---------|-------------|------|
| 모바일 (~640px) | 1열 | 풀 너비, 스택 레이아웃 |
| 태블릿 (640~1024px) | 2열 | 적절한 여백 유지 |
| 데스크톱 (1024px~) | 3~4열 | 최대 너비 1280px 제한 |

### 네비게이션

- **데스크톱**: 상단 고정 헤더 (로고 + 카테고리 + 검색 + 장바구니 + 로그인)
- **모바일**: 햄버거 메뉴 → 슬라이드 드로어
- **관리자**: 좌측 사이드바 네비게이션

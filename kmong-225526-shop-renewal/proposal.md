안녕하세요. 올려주신 프로젝트 내용을 검토하였습니다.

카페24, 아임웹 등 SaaS 빌더 기반 쇼핑몰을 탈피하여 클라우드 기반 자체 쇼핑몰을 구축하고, 기존 고객DB,상품DB를 이관하는 프로젝트로 이해했습니다.

유사한 이커머스 플랫폼을 기획부터 디자인, 개발까지 수행한 경험이 있습니다. Nuxt + Django + PostgreSQL 스택으로 상품 선택 > 에디터 디자인 > 결제까지 풀사이클 커머스를 구현했으며, 토스페이먼츠 결제 연동, 네이버/카카오 소셜 로그인, 카카오 알림톡 연동까지 포함합니다.

구현 계획 (19일):
1~2일 - DB 스키마 설계, 개발환경 구성
3~8일 - 백엔드 API + 프론트엔드 핵심 (상품, 회원, 장바구니)
9~10일 - 결제 연동 (PG사)
11~14일 - 주문/배송 관리, 관리자 페이지
15~16일 - DB 마이그레이션 (고객DB, 상품DB)
17~19일 - QA, 반응형 대응, 최종 배포

기술스택은 Next.js + Tailwind CSS + MySQL + Prisma로 제안드리며, 클라우드 배포로 호스팅 서버 없이 운영 가능한 구조입니다.

기존 고객DB,상품DB 이관은 이전 프로젝트에서 데이터 마이그레이션 스크립트를 작성하여 진행한 경험이 있습니다. 기존 빌더(카페24 등)의 API 또는 CSV 추출을 통해 데이터를 확보하고, Prisma 마이그레이션 도구를 활용하여 신규 MySQL DB에 매핑,이관하는 방식으로 진행합니다. 비밀번호 해시 호환이 불가한 경우 신규 비밀번호 발급 + 안내 메일 발송으로 대응합니다.

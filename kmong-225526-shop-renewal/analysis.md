## 2. 선정 기술스택

| 영역 | 기술 | 선정 이유 |
|------|------|----------|
| 프레임워크 | Next.js 14+ (App Router) | 프론트+백엔드 통합, SSR/SEO, 바이브코딩 생산성 최고 |
| 스타일링 | Tailwind CSS | 빠른 UI 개발, 반응형 기본 지원 |

| ORM | Prisma | 타입 안전성, MySQL 지원, 마이그레이션 도구 내장 |
| 인프라 | Vercel + PlanetScale(MySQL) | 서버리스/클라우드 기반, 운영 비용 최소화 |
| 결제 | 토스페이먼츠 or PG사 | 의뢰인과 협의 필요 |

## 8. 암묵적 필요 기능

> 쇼핑몰 프로젝트에서 명시적으로 나열되지 않았지만 필수적으로 필요한 기능들

| 기능 | 설명 |
|------|------|
| 상품 검색 / 필터 / 정렬 | 카테고리별, 가격순, 인기순 등 상품 탐색 기능 |
| 상품 카테고리 관리 | 관리자 페이지에서 카테고리 CRUD |
| 소셜 로그인 (네이버, 카카오) | 국내 쇼핑몰 필수 — 자체 회원가입 대비 전환율 향상 |
| 관리자 대시보드 (매출 통계, 주문 현황) | 일별/월별 매출, 주문 건수, 인기 상품 등 시각화 |

| 송장번호 관리 | 관리자가 송장번호 입력 → 구매자 마이페이지에서 조회 |
| 쿠폰/적립금 시스템 | 할인 쿠폰 발급, 적립금 적용 |
| 이메일/알림톡 알림 | 주문 확인, 배송 시작, 배송 완료 등 알림 발송 |

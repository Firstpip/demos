## 2. PFCT (PFC Technologies) 리브랜딩 홈페이지

- **사이트**: https://pfct.co.kr/
- **유형**: 금융 서비스 기업 홈페이지 (리브랜딩)
- **기술스택**: React + Framer Motion
- **인터랙티브 구현**:
  - 스크롤 기반 인터랙션 — 스크롤 시 "영상처럼 재생되는" 효과
  - `useTransform` + `useMotionValueEvent`로 스크롤 위치를 애니메이션에 매핑
  - 기하학적 심볼 애니메이션 (파이 차트 진행률 애니메이션)
  - 섹션별 액션 정의 — 누적 스크롤 비율에 따라 순차 실행
- **디자인 특징**: 기하학적 인터랙션, 풀페이지 대신 자유 스크롤
- **참고 포인트**: Framer Motion의 스크롤 매핑 기법, 섹션별 애니메이션 시퀀스 구조

## 3. 하이엔드 인터랙티브 기업 홈페이지 공통 패턴

### 페이지 구성
- **히어로 섹션**: 풀스크린 비주얼 + 타이포그래피 + 미묘한 모션
- **About/소개**: 기업 철학·비전을 은유적 이미지와 텍스트로 전달
- **서비스/역량**: 카드/그리드 레이아웃, hover 인터랙션
- **포트폴리오/프로젝트**: 그리드 갤러리 + 상세 페이지 전환 애니메이션
- **팀/파트너**: 인물 사진 + 역할 소개
- **문의/CTA**: 심플한 폼 또는 CTA 버튼

### 핵심 인터랙티브 기법
| 기법 | 설명 | 구현 도구 |
|------|------|----------|
| 스크롤 트리거 Fade-in | 섹션 진입 시 요소 페이드인 | GSAP ScrollTrigger / Framer Motion |
| 패럴랙스 | 레이어별 다른 속도로 이동 | GSAP / CSS |
| 텍스트 Split 애니메이션 | 글자가 한 글자씩 나타남 | GSAP SplitText |
| 스무스 스크롤 | 부드러운 관성 스크롤 | Lenis / GSAP ScrollSmoother |
| 페이지 전환 | 페이드/슬라이드/클립패스 전환 | Framer Motion AnimatePresence |
| 커서 인터랙션 | 커스텀 커서, 마그네틱 효과 | GSAP / CSS |
| 이미지 리빌 | 마스크/클립패스로 이미지 공개 | GSAP / CSS clip-path |
| 호버 확대/틸트 | 카드 hover 시 3D 틸트 | Framer Motion / CSS transform |

### 디자인 트렌드 (2025-2026)
- **깊이감 있는 질감**: 빛, 그림자, 블러, glassmorphism
- **다크 테마 + 강조 컬러**: 전문성과 고급감 표현
- **넉넉한 여백**: 콘텐츠 간 충분한 간격으로 고급감 연출
- **대형 타이포그래피**: 굵은 헤드라인으로 시선 집중
- **미니멀 네비게이션**: 햄버거 메뉴 또는 최소 항목
- **마이크로 인터랙션**: 버튼 hover, 스크롤 인디케이터, 로딩 효과

## 4. B2B IT/정보통신 기업 홈페이지 특징

- **컬러**: 다크 (검정/짙은 남색) + 포인트 컬러 (파랑, 시안, 보라)
- **톤**: 전문적, 신뢰감, 미래지향적
- **비주얼**: 추상적 3D 그래픽, 기하학적 패턴, 데이터 시각화 모티프
- **콘텐츠 구조**:
  - 서비스/솔루션 중심 레이아웃
  - 기술력 + 실적 수치 강조 (프로젝트 수, 클라이언트 수 등)
  - 케이스 스터디/포트폴리오
  - 파트너 로고 배열
- **인터랙션**: GSAP ScrollTrigger로 섹션별 동적 전환, 데이터 카운터 애니메이션

## 5. 데모 구현에 적용할 디자인 방향

### 컬러 팔레트 (가상 브랜드)
- **Primary**: 다크 (#0A0A0A ~ #1A1A2E)
- **Secondary**: 딥 블루 (#16213E)
- **Accent**: 시안/라이트 블루 (#0F969C 또는 #6C63FF)
- **Text**: 화이트 (#FFFFFF) + 라이트 그레이 (#A0A0A0)

### 폰트
- 한글: Pretendard (또는 SUIT)
- 영문: Inter 또는 Space Grotesk
- 대형 헤드라인: 700~900 weight

### 페이지 구성 (데모)
1. **메인 (히어로)**: 풀스크린 + 스크롤 인터랙션 + 은유적 메시지
2. **About**: 기업 소개 + 비전/미션 + 타임라인 애니메이션
3. **Services**: 서비스 영역 카드 + hover 인터랙션
4. **Portfolio**: 프로젝트 그리드 + 상세 모달/페이지
5. **Team**: 팀 소개 (선택)
6. **Contact**: 문의 폼 (Mock)

### 인터랙티브 요소 (구현 예정)
- 스크롤 트리거 페이드인/슬라이드인
- 히어로 섹션 패럴랙스
- 텍스트 스플릿 애니메이션 (헤드라인)
- 스무스 스크롤 (Lenis)
- 페이지/섹션 전환 애니메이션
- 카드 hover 틸트/확대 효과
- 스크롤 진행 인디케이터
- 반응형 햄버거 메뉴 (모바일)

# 유사 서비스 리서치: ARTWORK (예술인 채용·공연 플랫폼)

## 요약

선정 경쟁사 3개:
- **StageLync** (stagelync.com) — 동일 도메인, 해외 공연예술·댄서 직종별 채용 플랫폼
- **사람인** (saramin.co.kr) — 인접 도메인, 국내 1위 일반 채용 플랫폼 (공고에서 대비군으로 언급)
- **숨고** (soomgo.com) — 기능 특화, 국내 프리랜서·고수 매칭 플랫폼 (AI 견적·위치 기반 추천)

### 공통 패턴 3개 (이 프로젝트에 기본값으로 반영)
1. **상단 고정 검색바 + 카테고리 드릴다운** — 세 서비스 모두 홈 상단에 키워드 검색바와 카테고리 진입 동선을 병행. 리스트 필터는 지역·직무·경력 기준이 공통.
2. **공고·프로필 상세에서 우측 고정 CTA** — "지원", "스크랩", "견적요청" 같은 행동 버튼이 스크롤 중에도 고정 노출되어 이탈 없이 전환 유도.
3. **긴급성·개인화 배지 사용** — "D-day", "마감임박", "신규 공고", "위치 기반 추천" 같은 라벨을 카드마다 붙여 탐색 속도와 클릭률을 끌어올림.

### 차별화 기회 2개 (이 프로젝트에서 다르게 갈 지점)
1. **예술인 특화 포트폴리오 표시** — 일반 채용 플랫폼(사람인)은 텍스트 이력서 중심, 프리랜서 플랫폼(숨고)은 시공 사례 사진 중심. ARTWORK는 **영상·공연 사진·장르 태그·경력 하이라이트**를 통합한 프로필 카드를 전면에 둠으로써 댄서·예술인 특성 반영 (StageLync도 직종 태그만 있고 영상 일원화는 부족).
2. **채용공고 + 프로젝트/공연 모집의 이중 게시판** — 기업 주도 채용(상용 계약)과 개인·팀 주도 프로젝트·공연 멤버 모집을 같은 동선에서 제공. StageLync/사람인은 기업 공고 단방향, 숨고는 의뢰자 단방향. ARTWORK는 두 흐름을 동시에 지원하는 구조가 차별 지점.

## 후속 검토 메모

- StageLync의 국제 공고 "비자·거주지" 영역을 국내판으로 번역: **공연 조건 (제공 항목·의상·교통·식사·연습 일정)** 영역을 공고 상세에 배치.
- 사람인의 상품 배너는 유료 모델이므로 ARTWORK MVP에는 "추천 공고"로 단순화 (광고 구조는 Phase 2에서 판단).
- 숨고의 에스크로(숨고페이)는 MVP 범위 밖. 다만 "지원 후 섭외 안전 가이드" 같은 텍스트 영역으로 신뢰 신호 일부 차용 가능.

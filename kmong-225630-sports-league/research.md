# 유사 서비스 리서치: 스포츠리그 경기신청 및 대진표 플랫폼

## 1. BKPLAY (배드민턴 대진표)

- 사이트: https://www.bkplay.kr / https://sfa.bkplay.kr
- 유형: 한국배드민턴협회 공식 대진표 플랫폼

### 특징
- 3개 섹션 분리 운영: 생활체육, 승강제리그, 유청소년리그
- 대회 상태 5단계: 접수중, 접수예정, 접수완료, 진행중, 종료
- 전국 선수 109,474명, 팀 3,828팀 데이터 보유
- 지역별 필터링 + 지도 인터랙션 통계

### 페이지 구조
- 대회정보: 전체 일정, 지역별 정보
- 팀/선수정보: 팀/선수 조회, 지역통계
- 미디어센터: 공지사항, 뉴스, 영상

### UI/디자인
- 상태별 컬러 배지 (연핑크, 연노랑, 연파랑)
- 그리드 기반 카드형 레이아웃
- Swiper 슬라이더
- 모달 기반 선수 상세정보
- PC/모바일 별도 템플릿

### 참고 포인트
- 대회 상태 배지 시스템 (5단계) → 우리 프로젝트 대회 상태 관리에 참고
- 선수 프로필 (성별, 출생년도, 급수) → 카테고리별 인증에 참고
- 지역별 필터링 + 검색 → 대회 검색 UI에 참고
- 팀/선수 통계 대시보드 → 랭킹 페이지에 참고

## 2. 위꾹라이브 (배드민턴 대회 운영 플랫폼)

- 사이트: https://wekkuklive.com
- 유형: 배드민턴 대회 운영 SaaS

### 특징
- AI 기반 99% 자동화 (대진표 자동생성, 스케줄링)
- MatchBoard25: 25분 내 대진표 완성
- 실시간 라이브 스코어링
- 심판용 앱 (원터치 스코어 입력)
- 카카오톡 자동 알림

### 페이지 구조
- 서비스 소개, 이용 요금, 영상서비스 활용법
- 견적계산기, 문의하기
- 로그인/회원가입

### UI/디자인
- 컬러: Primary 녹색(HSL 142, 76%, 36%), Secondary 주황(HSL 24, 95%, 53%)
- 반응형 디자인 (모바일/데스크톱)
- 그리드 기반 갤러리 (4열)
- 부드러운 스크롤 애니메이션
- 배경 블러 효과 (backdrop-blur)

### 참고 포인트
- 견적계산기 UI → 예산 투명성 컴포넌트 참고
- 소셜 로그인 (구글, 카카오, 네이버) → 인증 UI 참고
- 대회 상태별 자동 알림 → 푸시 알림 시나리오에 참고
- 녹색+주황 컬러 조합 → 스포츠 느낌의 디자인 톤에 참고

## 3. BracketHQ (토너먼트 대진표 메이커)

- 사이트: https://brackethq.com
- 유형: 범용 토너먼트 대진표 생성 플랫폼

### 특징
- 싱글/더블 엘리미네이션, 라운드 로빈 지원
- 테마 커스터마이징
- 실시간 점수 업데이트
- 무제한 대진표 크기
- 참가자 시드 배정

### 참고 포인트
- 대진표 시각화 방식 (트리 구조) → 토너먼트 대진표 UI에 참고
- 싱글/더블 엘리미네이션 토너먼트 형식 → 대진표 알고리즘에 참고
- 라운드 로빈 리더보드 → 리그전 랭킹 표시에 참고

## 4. Challonge (토너먼트 관리 플랫폼)

- 사이트: https://challonge.com
- 유형: 글로벌 토너먼트 관리 플랫폼

### 특징
- 싱글/더블 엘리미네이션, 라운드 로빈, 스위스 방식 지원
- 임베더블 대진표 위젯
- 커뮤니티 기반 토너먼트
- API 제공

### 참고 포인트
- 다양한 토너먼트 형식 지원 → 향후 확장 시 참고
- 대진표 임베드 기능 → 공유 기능에 참고

| 푸시 알림 | 위꾹라이브 (카카오톡 알림) |

## 디자인 분석 종합

### 컬러 트렌드
- BKPLAY: 회색 기본톤 + 상태별 파스텔 컬러 배지
- 위꾹라이브: 녹색(Primary) + 주황(Accent) — 스포츠 에너지 느낌

### 추천 디자인 방향
- Primary: 블루 또는 녹색 계열 (스포츠 신뢰감)
- Accent: 주황 또는 노랑 (에너지, 활동성)
- 상태 배지: 파스텔 컬러 시스템 (접수중-파랑, 진행중-녹색, 종료-회색 등)
- 카드형 레이아웃 + 그리드
- 대진표: 트리 구조 시각화 (좌→우 흐름)
- 반응형: 모바일 우선 (앱 기반 서비스)

안녕하세요. 프로젝트 내용 꼼꼼히 확인했습니다.

아임웹 기반 도매몰을 B2B 폐쇄몰로 자체구축 전환하는 프로젝트로 이해했습니다.

핵심 요구사항을 정리하면:

B2B 폐쇄몰 — 로그인/회원가입 후에만 사이트 접근 가능
사업자 전용 회원체계 — 사업자번호 검증, 사업자등록증 + 소매인지정서 업로드, 사업자 및 본인(성인) 이중 인증, 관리자 승인 후 이용
상품관리 — 옵션별 재고 설정, 회원별 상품 공개 여부 설정
주문관리 — 구매 기준 제한(금액/수량/기간), 필터 선택 엑셀 다운로드, 관리자 직접 주문 생성, 주문 확정 전 수정
결제 — 계좌이체/무통장 입금만 (PG 불필요)
회원 혜택 — 회원별 할인, 현금영수증 번호 자동 적용 (개인/법인)
외부 연동 — 이카운트 ERP(주문조회, 이벤트성 상품), 회계솔루션 API(세금계산서/현금영수증 자동발행)
관리자 — 대시보드, 팝업 관리(단일/슬라이드)
DB 마이그레이션 — 기존 아임웹 데이터 이전
모바일 반응형

유사한 이커머스 플랫폼을 기획, 디자인, 개발까지 1인 풀스택으로 수행한 경험이 있습니다. 상품 관리, 주문/결제 플로우, 회원 시스템, 외부 API 연동 등 본 프로젝트의 핵심 기능과 직접적으로 연관됩니다.

구현 계획:
1주차 — 프로젝트 세팅 + DB 설계 + 회원 시스템(사업자 가입, 이중인증, 관리자 승인) + 상품관리
2주차 — 주문/결제 + 회원별 할인/현금영수증 + 관리자 패널(대시보드, 팝업, 주문생성)
3주차 — 이카운트 ERP 연동 + 회계솔루션 연동 + DB 마이그레이션(아임웹) + 반응형 + QA/배포

기술스택: Next.js 14 + Tailwind CSS + PostgreSQL + Prisma (AWS Lightsail + RDS + S3)

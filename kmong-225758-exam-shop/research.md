# 유사 서비스 리서치

## 조사 대상

### 1. 마더텅 인터넷서점 (book.toptutor.co.kr)
- **유형**: 모의고사 교재 전문 출판사 직영 서점
- **특징**: 학년별(초/중/고) + 시리즈별 이중 분류 체계
- **페이지 구성**: 메인(배너+베스트+시리즈), 카테고리별 목록, 상품 상세, 장바구니, 주문
- **디자인**: 심플한 교육 톤, 흰 배경 기반, 시리즈별 컬러 코드(까만책/빨간책/노란책)
- **주요 기능**: 검색(자동완성), 할인율/적립금 표시, 대량주문 안내
- **참고 포인트**:
  - 시리즈별 컬러 코드 → 모의고사 과목별 컬러 분류에 활용
  - 학년별 + 시리즈별 이중 필터 → 과목별 + 유형별 이중 필터에 적용
  - 동영상강의 연계 → 자료실 다운로드와 유사한 부가 서비스 패턴

### 2. 한국교과서협회 쇼핑몰 (ktbookmall.com)
- **유형**: 교과서/지도서 전문 온라인 쇼핑몰
- **특징**: 학교급별 3단계 분류, 일반/단체 주문 구분
- **페이지 구성**: 메인(학교급 그리드), 일반주문, 단체주문, 장바구니, 주문/배송조회, 고객센터
- **디자인**: 흰 배경 + 회색 아이콘, 기관 느낌의 깔끔한 톤
- **주요 기능**: 검색, 장바구니(일반/단체 분리), 주문 추적, 반품 신청, 비공개 Q&A
- **참고 포인트**:
  - 단체주문 기능 → 학교/학원 대량 구매 시나리오 참고
  - 개정 연도 병기 표시 → 모의고사 연도/버전 관리에 활용
  - 반품 정책 명시 → 주문 관리에 반품/환불 프로세스 포함

### 3. 용봉서적 (shop.10gongpass.com)
- **유형**: 수험서적 전문 서점
- **특징**: 시험 유형별 카테고리 (공무원, 교원임용, 자격증 등)
- **페이지 구성**: 메인(배너+추천+신간+후기), 카테고리 목록, 상품 상세, 장바구니
- **디자인**: 네이비+화이트 기반, 오렌지/레드 강조색, 전문적/학습 톤
- **주요 기능**: 검색(상품명/저자/ISBN 필터), 인기 키워드, 장바구니/바로구매, 정오표/추록 게시판
- **참고 포인트**:
  - 네이비+화이트 배색 → 교육/출판 느낌의 신뢰감 있는 컬러
  - 정오표/추록 게시판 → 자료실 기능의 활용 사례
  - 검색 필터(상품명/저자/ISBN) → 모의고사의 과목/학년/출판연도 필터에 적용
  - 후기 섹션 메인 노출 → 구매 전환에 효과적

## 기능별 참고 매핑

| 기능 | 참고 서비스 | 적용 방안 |
|------|-----------|----------|
| 상품 카테고리/필터 | 마더텅 (이중 분류) | 과목별 + 학년별 이중 필터 |
| 상품 리스트 | 용봉서적 (그리드+가격) | 썸네일 + 할인가 + 적립금 표시 |
| 검색 | 용봉서적 (멀티필터) | 상품명/과목/학년 복합 검색 |
| 장바구니/주문 | 한국교과서협회 | 수량 변경, 주문 추적 |
| 후기 | 용봉서적 (메인 노출) | 상품 상세 + 메인 페이지 노출 |
| 자료실 | 용봉서적 (정오표/추록) | 교재별 보충자료 다운로드 |
| 배송 관리 | 한국교과서협회 | 주문/배송 조회, 반품 신청 |
| 관리자 | 공통 패턴 | 상품/주문/회원 CRUD + 대시보드 |
| 디자인 톤 | 용봉서적 | 네이비+화이트, 전문적/학습 톤 |

## 데모 디자인 방향

- **컬러**: 네이비(#1B2A4A) + 화이트 기반, 교육적 신뢰감
- **강조색**: 오렌지/코랄(#E8653A) - CTA 버튼, 할인 표시
- **서브컬러**: 밝은 그레이(#F5F5F5) - 섹션 구분 배경
- **폰트**: Pretendard (한글 가독성 최적)
- **레이아웃**: 깔끔한 그리드, 여백 넉넉한 교육 콘텐츠 스타일
- **톤**: 전문적이면서 접근하기 쉬운 교육 플랫폼 느낌

## 1. 채널톡 (https://channel.io/ko) — 메인 벤치마킹

### 섹션 순서

1. 헤더 — 로고 / 제품 / AI 에이전트 / 가격 / 고객사례 / 리소스 / 로그인 / 무료 시작
2. 히어로 — 좌측 카피 "고객상담의 미래는 AI입니다" + CTA 2개 / 우측 대시보드 스크린샷 (그라데이션 배경)
3. 고객사 로고 슬라이더 ("다양한 산업과 규모의 조직이 선택한 업무 플랫폼")
4. 통계 섹션 — "6만+ / 63만+ / 688만+" 수치 강조
5. 메인 메시지 — "채널톡은 고객과 팀을 위한 올인원 AI 메신저입니다"
6. 기능 카드 그리드 — 6개 기능 (메신저/전화/팀메신저/워크플로우/도큐먼트/마케팅), 파스텔 톤 카드
7. ALF (AI 에이전트) 소개 — 자사 AI 제품 단독 섹션
8. 동영상 + 고객 증언
9. 구독 플랜 (월간 가격, 2개 플랜 비교)
10. 보안 인증 섹션 — 다크 블루 배경, 인증 배지/키 아이콘
11. G2 평점 / 도입 성과
12. 최종 CTA — 그라데이션 배경 "무료로 시작하기"
13. 푸터

### 디자인 시스템

| 항목 | 내용 |
|------|------|
| 히어로 그라데이션 | 라이트 퍼플/블루 그라데이션 (연한 톤) |
| 보안 섹션 컬러 | 다크 블루 (#1a1f3a ~ #0f172a 추정) |
| 최종 CTA | 블루→퍼플 그라데이션 |
| 기능 카드 | 파스텔 핑크/옐로우/민트/라벤더 각각 다른 컬러 |
| 폰트 | Pretendard 계열 (한국 B2B 표준) |
| CTA 스타일 | Primary(블루 솔리드) + Secondary(아웃라인) |

### SafeWorks에 가져올 핵심 패턴

- **히어로 좌측 카피 + 우측 대시보드**: 그대로 차용 (카피만 SafeWorks 톤으로)
- **고객사 로고 슬라이더**: 영신디엔씨 + 건설사 로고로 교체
- **통계 수치 섹션**: "도입 현장 수 / 관리 인원 / 안전 이벤트 처리 건" 등 건설 안전 도메인 KPI로 재구성
- **기능 카드 그리드**: SafeWorks 핵심 기능 4~6개로 재구성 (작업자 안전 / 출입 관리 / 현장 대시보드 / 알림 등)
- **보안 섹션 다크 블루**: 그대로 차용 (엔터프라이즈 신뢰도)
- **최종 CTA 그라데이션**: 그대로 차용

### 차별화 포인트 (표절 방지)

- 채널톡은 대화형 메시저 UI가 핵심 → SafeWorks는 건설 현장 대시보드/안전 관리 UI로 교체
- 컬러 팔레트는 유지하되 포인트 컬러를 **세이프티 옐로우(#FFCC00 계열)** 또는 건설 안전 아이콘 톤으로 조정
- 카피 톤은 "대화" → "안전", "상담" → "관리", "AI 에이전트" → "실시간 모니터링" 등 건설 IT 도메인으로 완전 재작성

## 2. Dooray (https://dooray.com/main/) — 롤링 애니메이션 레퍼런스

### 전체 섹션

- 히어로 — "이제 금융사도 구축하지 않고 SaaS Dooray!를 사용할 수 있습니다"
- 제품 기능 소개 (스크롤 연동 롤링/캐러셀)
- 보안 인증 배지
- 고객사 소개
- CTA
- 푸터

### 롤링 애니메이션 구현

- 12개 서비스 카드를 두 번 반복
- Prev/Next 버튼 + 페이지 인디케이터 (0,1,2,3,4)
- 스크롤 또는 좌우 슬라이드 전환
- 각 서비스가 순차 전환되며 화면 중앙에 대시보드 스크린샷 노출

### SafeWorks에 가져올 패턴

- **세이프웍스 기능소개 (롤링)** 섹션에 동일 패턴 적용
- GSAP ScrollTrigger로 구현 권장 (스크롤 픽시트 + 좌측 설명/우측 이미지 전환)
- 4~6개 기능 로테이션 (작업자 안전 / 출입 관리 / 알림 / 분석 등)

## 디자인 시스템 결론

**컬러 팔레트**
- Primary: `#2B6CDF` (건설 안전 블루)
- Secondary: `#FFCC00` (세이프티 옐로우, 포인트 CTA/경고)
- 히어로 그라데이션: `#E0E7FF → #DBEAFE → #FEF3C7` (라이트 블루/옐로우 믹스)
- 보안 섹션: `#0F172A` (다크 네이비)
- 최종 CTA 그라데이션: `#2B6CDF → #7C3AED`
- 기능 카드 배경: 파스텔 핑크/민트/라벤더/옐로우 (채널톡식 다채로움)

**폰트**
- Pretendard (한국 B2B 표준, 가독성)

**레이아웃 원칙**
- 히어로만 그라데이션 + 대시보드 스크린샷
- 본문은 화이트/그레이 배경 유지하며 섹션마다 subtle 구분
- 보안 섹션은 유일한 다크 영역 (대비 강조)
- 최종 CTA에서 다시 그라데이션으로 수미상관

**인터랙션**
- 헤더: 스크롤 시 배경 블러/섀도우 추가
- 로고 슬라이더: Swiper.js autoplay (방향: 좌→우)
- 기능 롤링: GSAP ScrollTrigger (좌측 설명 고정 + 우측 이미지 전환)
- 플로팅 배너: 스크롤 500px 이후 표시, 우측 하단 고정
- 카드 hover: subtle lift + shadow
- 동영상: YouTube 임베드를 모달로 오픈

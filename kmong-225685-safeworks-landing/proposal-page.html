히어로 배너 (대시보드 UI + 서비스 이용하기/견적 문의)

고객지원 (FAQ 아코디언 + 도입문의 Form)

플로팅 배너 (브로슈어 다운로드 퀵 버튼)
모바일 반응형 웹디자인
핵심 역량

섹션 순서, CTA 라벨("서비스 이용하기"/"견적 문의"), 참고 사이트, 폼 필드까지 본문 그대로 구현. 임의 해석 없음.

의뢰 명시 스택 100% 준수

HTML, CSS, JavaScript, Node.js, MySQL, Figma. 추가 프레임워크 없이 인수인계와 유지보수가 쉬운 네이티브 구조.

실제 동작 데모 사전 제공

제안 단계에서 메인 8섹션 + 서브 4페이지 + 30개 인터랙션이 동작하는 데모를 GitHub Pages에 미리 배포했습니다.

인수인계 + 6개월 무상 유지보수

코드·DB 스키마·배포 가이드까지 정리해 인계해 드리고, 납품 후 6개월간 발견되는 이슈는 무상으로 수정해 드립니다.

SafeWorks 관점에서의 강점
콘텐츠 분석크몽 본문·Figma 기획서·영신디엔씨 공식 영상까지 분석해 SafeWorks가 "안전 + 문서 관리 통합" 솔루션임을 정확히 파악. 카피·UI 목업·기능 설명에 일관 반영.
참고 사이트메인 배너는 마키나락스, 제품 소개는 Dooray sticky 롤링, 요금 안내는 무스마 우측 반복 CTA로 의뢰인이 짚어준 3개 사이트 패턴 그대로 reproduce.
핵심 카피영상의 핵심 메시지 "안전은 대응이 아닌, 예방입니다"를 히어로 카피로 채택. "현장은 안전하게, 관리자는 편리하게, 기업은 탄탄하게" 최종 CTA에 반영.
요구사항폼은 의뢰 명시 그대로 소속 / 이름 / 연락처 / 이메일 / 문의 내용 5필드. 임의 추가 없음. 관리자 메일 발송은 Node + Nodemailer로 구현.
옵션 안내의뢰인 요청대로 풀버전(2,700,000원)과 원페이지(2,000,000원) 두 옵션의 가격 차이를 명시. 부가세 별도, 세금계산서 발행 가능.
품질데스크톱(1440) / 태블릿(768) / 모바일(375) 3종 뷰포트와 30개 인터랙션을 Puppeteer로 자동 검증해 매일 동작 보장.
